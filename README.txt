To startup call answering server locally:

In Terminal 1:
```
cd Scripts
Activate
cd ..
cd Twilio
python answer_call.py
```

In Terminal 2:
```
ngrok http 5000
```
copy the https url generated by ngrok
under the phone settings for your number on Twilio, go to Voice & Fax -> A Call Comes In (Webhook) -> paste the url
save changes
within 5 minutes server should be set up
